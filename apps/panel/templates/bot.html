Setup Guide

Configure your bot with @BotFather

1. Get Bot Token

Open @BotFather and send /newbot. Copy the HTTP API Token and paste it in the Settings above.

2. Configure Mini App

To enable the "Subscription Page" (Mini App), map the menu button to your panel's domain:

A Send /mybots -> Select your bot
B Select Bot Settings -> Menu Button -> Configure Menu Button
C Send the URL of your panel (must be HTTPS!):
https://YOUR_DOMAIN/

3. Enable Inline Mode

Go to Bot Settings -> Inline Mode -> Turn on. This allows sharing subscriptions via chat.
